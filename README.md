# MOOC & Open Source Group

>东岳 Mos 组

## 活动

### 周期性活动

1. 一个月一次的成员分享（题材内容不限，只要与 Programming 有关）
2. 以课程为单位的分享（视课程进度而定）

### 不定期活动

1. 出 idea，实现并开源，力求小而美 ([Idea 承包指南](./idea-intro.md))
2. 论文阅读活动
3. 相关开源项目技术分享

## 招新

长期有效

- 基础要求
  - 至少了解一门编程语言 (如 Java, C++ 等);
  - 想为开源做贡献.
- 加分项
  - 自己维护有开源项目, 或者是开源项目的 contributor;
  - 熟悉一个或多个开源项目.

注: 因为 Mos 组是今年才设立的, 所以各方面都不是很成熟, 如果有大腿还望不吝赐教!
